```

        ↓
挖洞过程（Claude + 工具）
        ↓ loop结束

     ↓ 每条记录立即升级 Payload 库

├── Step 1: H1 Sync — 拉取最新 triage 结果

├── Step 2: 失败分类分析 — 每种失败触发进化动作
├── Step 3: 技法原子重组 — 成功技法拆解→衍生N变体
├── Step 4: 知识蒸馏 — Payload 库升级
├── Step 5: Benchmark 盲区 — 量化差距 + 104 CTF 靶机验证
├── Step 5.5: 架构差距补全 — Agent拆分/企业身份矩阵/Brain持久化
└── Step 6: 推荐引擎 — 生成本周优先方向
        ↓
下一个目标：方法论更强，命中率更高
```

```bash
# 半自主挖洞（全自动记录）
python aimy.py --autopilot --target target.com --max-targets 5 --budget 3600

# 手动飞轮（完整6步）
python aimy.py --flywheel --skip-h1

# 量化 vs XBOW 差距
python run_benchmark.py --tags ssrf --limit 10

# 交互模式（自动记录钩子已内置）
python aimy.py -t target.com
```
