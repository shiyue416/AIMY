## 常用命令速抄

```bash
# 子域名发现 → 探活
subfinder -d target.com -silent | httpx -silent -title -status-code -tech-detect

# 历史URL收集（双管齐下）
gau target.com --subs > urls.txt
waybackurls target.com >> urls.txt

# 参数筛选
cat urls.txt | gf ssrf
cat urls.txt | gf sqli
cat urls.txt | gf redirect

# nuclei 全扫
nuclei -u https://target.com -t ~/nuclei-templates/ -severity critical,high -o results.txt

# Secret 泄露
trufflehog github --org=target --only-verified

# 目录Fuzz
ffuf -u https://target.com/FUZZ -w ~/SecLists/Discovery/Web-Content/big.txt -mc 200,301,302,403 -t 50

# JS文件收集
getJS --url https://target.com --complete

# OOB 监听
interactsh-client -server oast.pro

# 递归深度侦察
"C:/Users/PC/AppData/Local/Programs/Python/Python312/Scripts/bbot.exe" -t target.com -p subdomain-enum
```
