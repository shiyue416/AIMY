## waybackurls

- **链接**: https://github.com/tomnomnom/waybackurls
- **作者**: tomnomnom（gf 同作者）
- **与 gau 的区别**:

| | gau | waybackurls |
|--|-----|------------|
| 数据源 | Wayback + OTX + CommonCrawl | 仅 Wayback Machine |
| 速度 | 较慢（多源） | 更快 |
| 覆盖率 | 更广 | 专精 Wayback |
| 用法 | 日常首选 | gau 补充/备用 |

- **用法**:
```bash
waybackurls target.com | tee wayback_urls.txt
# 双管齐下
gau target.com --subs > all_urls.txt
waybackurls target.com >> all_urls.txt
sort -u all_urls.txt > dedup_urls.txt
```

## getJS

- **链接**: https://github.com/003random/getJS
- **唯一价值**: 专门收集目标所有 JS 文件 URL，比 katana 更轻量
- **与 katana 的区别**:

| | katana | getJS |
|--|--------|-------|
| 功能 | 全站爬取+JS渲染 | 只收集JS文件URL |
| 速度 | 较慢 | 极快 |
| 场景 | 全面爬取 | 快速获取所有JS列表 |

- **用法**:
```bash
# 收集目标所有JS文件
getJS --url https://target.com --complete

# 配合管线提取端点
getJS --url https://target.com --complete | xargs -I{} curl -sk {} | grep -oP '(?<=["'"'"'])/[a-zA-Z0-9_/?=&.-]+'
```
